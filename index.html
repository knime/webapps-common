Show Description.vue source code 
<script>

export default {
    props: {
        /**
         * the text to be shown
         */
        text: {
            type: String,
            default: null
        },
        /**
         * whether the provided `text` should be rendered as HTML or plain text
         */
        renderAsHtml: {
            type: Boolean,
            default: false
        }
    }
};
</script>

<template>
  <div
    v-if="$slots.default"
    class="description"
  >
    <!-- @slot if content is provided via slot, the `text` and `renderAsHtml` props will be ignored -->
    <slot />
  </div>
  <!-- eslint-disable vue/no-v-html -->
  <div
    v-else-if="!$slots.default && renderAsHtml"
    class="description"
    v-html="text"
  />
  <div
    v-else
    class="description plain"
    v-text="text"
  />
</template>

Grid System

The grid system consists of 12 columns. You can use it by nesting a <section> and a .grid-container with up to twelve .grid-items as shown below.

The section stretches across the whole screen, and can be used for background-colors. The grid container limits the width of the actual content. The grid items need not necessarily add up to 12 columns.

Grids items can be nested in another grid container.

Responsiveness

The grid has a responsive design. Therefore two breakpoints are set. This reduces the number of columns the consists of to four (at 1180px) and two (at 900px) respectively. Grid items will be grouped together equally to fit the scheme.
