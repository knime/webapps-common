The Pill component has been enhanced to include five new status variants: Success, Info, Warning, Error, and Promotion. These variants provide additional ways to convey different statuses or messages, improving the overall user experience and visual communication within the interface. Users can now interchange icons and text within the Pill, allowing for a more personalized presentation. Furthermore, The Pill component now allows users to customize the icon style for both the Default and Muted variants, offering enhanced flexibility in design.
